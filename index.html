Networks and communication Network

Collection of computers connected to one another through the communicatuion media to share the information.

Network topologies

The particular way all the devices such as computers, printers, servers etc.are connected to each other is known as TopologyThere are several topologies are used .some of the most popular once are.


Bus Topology

In a bus topology, all nodes in the network are connected directly to a central cable that runs up and down the network - this cable is known as the backbone.
Data is sent up and down the backbone until it reaches the correct node.

Advantages
  1. Simplicity
  2. Cost effective ness
  3. Efficient for small network
    4.

Disadvantages

  1. It is not suitable for large networks.
  2. It is difficult to identify and troubleshoot problems if the whole network goes down.
  3. It requires terminators at both ends of the main cable.
  4. It has limited security options and data quality.
  5. It slows down the network when additional devices are added.
  6. It can cause an entire network to collapse if there is a break in the backbone.
  7. It can have bus termination issues that affect the network performance.
    8.
Star Topology

In this topology all the computers are connected to central HUB.All the data has to be routed through the hub and the hub controls the data traffic in the network.Data transmission is faster.It helps to implement security measures.

Advantages

  1. Reliability
  2. Security
  3. Ease of installation
    4.

Disadvantages

  1. It requires more cable and hardware than other topologies, making it more expensive and prone to damage.
  2. It depends on the central device (hub or switch) to function, and if it fails, the whole network goes down.
  3. It has limitations of connections and speed due to the bandwidth of the central device.
  4. It reduces the mobility of the nodes and makes it difficult to attach mobile devices to the network.
    5.
Tree topology

A bus and star network topology combined to form a tree topology is known in computer networks. This topology's key benefits include being very flexible and having superior scalability. With only one route between any two network nodes, the tree network topology is thought to be the simplest topology of all the topologies.

Advantages

  1. Hieralchical Structure
  2. Scalability
  3. Easy Maintanence
    4.

Disadvantages

  1. Configuration Complexity
  2. Backbone Vulnerability
  3. Propagation of Errors
  4. Cabling Requirements
    5.  Apart from this ring topology,mesh topology,hybread topology are some other topologies.
